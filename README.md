FlixZone

A modern streaming platform where you can browse movies and TV shows, create an account, and watch content with a smooth, intuitive interface. Built like Netflix with a clean UI and fast performance.

--- WHAT IT DOES ---

FlixZone is a video streaming application that lets users discover and watch entertainment. You can sign up, log in securely, browse through a catalog of content organized by categories, and play videos in a built-in player. Everything is responsive and works great on any device.

--- TECH STACK ---

React 19.0 - JavaScript UI framework
Vite 6.2 - Fast build tool
Firebase - Authentication and database (Firestore)
React Router 7.2 - Navigation between pages
react-firebase-hooks - Firebase integration
CSS - Styling

--- HOW IT WORKS ---

User Flow
1. User lands on the home page and sees featured content
2. User clicks "Sign In" to create an account or log in with email/password
3. Authentication is handled by Firebase
4. Once logged in, user can browse the content library
5. User clicks on a movie/show title card
6. Video player page loads with the selected content
7. User can watch with standard playback controls

--- PAGES IN THE APP ---

Home - Browse all available movies and shows
Login - Sign up or log in with email and password
Player - Watch the selected video with player controls

--- COMPONENTS ---

Navbar - Top navigation bar with user menu
Footer - Footer section with links
Titlecards - Individual movie/show cards with thumbnails
Home Page - Featured content and browsing grid
Login Page - Authentication form
Player Page - Video player with controls
